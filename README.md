# week-4
| Concept / Property                                  | Where I Used It In The Project                                                                                                                    |
| --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Text Properties (color, shadow, transform, spacing) | Hero title (text-shadow, text-transform), About paragraph (letter-spacing, word-spacing)                                                          |
| Inheritance                                         | font-family set on body                                                                                                                           |
| Google Fonts                                        | Cairo font from Google Fonts                                                                                                                      |
| Cursor                                              | cursor: pointer on links                                                                                                                          |
| Float + Clear                                       | Profile image floated left in About section                                                                                                       |
| calc()                                              | Profile image width using calc()                                                                                                                  |
| Opacity                                             | Navigation bar opacity                                                                                                                            |
| Position + Z-index                                  | Fixed navigation bar and Back-to-top button                                                                                                       |
| List Styling                                        | Skills list with removed bullets and custom icon using ::before                                                                                   |
| Table Styling                                       | Styled table for Experience/Education                                                                                                             |
| Pseudo-classes                                      | Hover effect on links                                                                                                                             |
| Pseudo-elements                                     | ::before for skills list, ::after for heading decoration, ::first-letter and ::first-line in About section, ::selection for custom text selection |
